================================================================================
USAGE EXAMPLES
================================================================================

For End Users (Installation):
```bash
# Marketplace installation
claude plugin marketplace install shuhe-work-skills

# Local installation
claude plugin install /path/to/work-skills

# Verify installation
/skills list
/guanyuan-data-fetcher --card-id 12345
```

For Developers (Contributing):
```bash
# Setup development environment
git clone https://github.com/shuhe/work-skills.git
cd work-skills
python -m venv venv
source venv/bin/activate
pip install -r requirements-dev.txt

# Create new skill
mkdir -p skills/my-new-skill/tests
cp templates/SKILL.md skills/my-new-skill/
cp templates/main.py skills/my-new-skill/

# Run tests
pytest tests/ -v --cov

# Submit PR
git push origin feature/add-my-new-skill
```
